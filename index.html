Projects
My major projects interest is in Consumer Development, Data Management, Data Mining and High-Dimensional Databases.

 Current:
  • WaterSafeDevice - a smart bracelet for preventing children drowning, Skoltech (github)
    This device is a bracelet to be worn on the child's hand and be followed with data processing from accelerometer, gyroscope and pulse oximeter by machine learning methods. In case data from sensors match dangerous pattern, device is sending alarm signal to parents or instructor key ring and to all smart bracelets located near the child.

  • Gesture control system for Unity (pdf, github)
    Andrey Kolchanov, Vladimir Eremin
    The purpose of this research is developing a mechanism of creation custom continuous gestures for Senz3D camera. The project allows manipulating (move, zoom, rotate, select parts, cut) 3D model of human body in Unity game engine using gestures; and will use the Oculus Rift helmet to create a virtual reality of surgeon operations. This application will be used by surgeons for education purposes.
